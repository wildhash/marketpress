## How to Capture

1. Deploy MarketPress to Hex
2. Click "Run All" and wait for data to load
3. Take full-screen screenshots of each view
4. Save with the filenames above
5. Ensure screenshots show the newspaper-style layout clearly

## Image Requirements

- Format: PNG (for clarity)
- Size: 1200-1600px wide (for README display)
- Quality: High resolution for professional appearance
- Content: Real market data (not dummy/placeholder data)
